# 心灵共鸣舱 - 一个温暖的情感支持应用

## 项目简介
心灵共鸣舱是一个专注于情感支持的应用，旨在为用户提供一个温暖、安全的情感表达和交流空间。

## 功能特点
- **情感表达**：通过多种方式表达和记录自己的情感
- **放松减压**：提供呼吸练习、冥想等放松方式
- **AI支持**：智能情感分析和支持
- **情绪日记**：记录和跟踪情绪变化

## 项目结构
```
心灵共鸣舱/
├── Pages/              # 页面目录
│   ├── AI/            # AI相关页面
│   ├── Evaluate/      # 评估相关页面
│   ├── MainPage/      # 主页面
│   ├── Mine/          # 个人中心
│   └── Release/       # 释放压力页面
├── static/            # 静态资源
├── Index.html         # 首页
└── README.md          # 项目说明
```

## 技术栈
- HTML5
- CSS3
- JavaScript
- 前端框架：原生JavaScript
- 后端：Node.js（AI模块）

## 如何使用
1. 克隆项目到本地
2. 打开Index.html文件即可访问应用
3. 对于AI功能，需要在Pages/AI目录下运行 `npm install` 安装依赖，然后运行 `node server.js` 启动服务
